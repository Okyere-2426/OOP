# OOP
Special methods (dunder methods)

1. constructor method (__init__()):
this method is used to initialize attributes of an object

2. 1.  informal string representation methods (__str__())
it defines what should be displayed when an object is printed

2. 2. formal representation (__repr__())
it provides an official or developer-oriented representation of an object

3. length method(__len__())
returns the length of a list

4. equality method (__eq__())
used for comparing objects

5. operator overloading(__add__(),__sub__() etc)
helps users to perform mathematical operations

6. item access (__getitem__())
allows elements to be accessed using indexing notation

7. item assignment (__setitem__())
allows assignment using indexing

8. membership testing (__contains__())
used to check the presence or absence of an item in a collection

9. callable objects (__call__())
makes an object behave as a function

10. iteration support (__iter__(),__next__())
allows objects to work in loops or iteration

11. boolean evaluation (__bool__())
allows objects to define what should be considered True False

12. destructor method (__del__())
 executed when an object is about to be removed or destroyed from memory
